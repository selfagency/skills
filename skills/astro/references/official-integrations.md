# Official Astro Integrations Reference

Quick config map for common `@astrojs/*` integrations.

## Framework integrations

- `@astrojs/react`
- `@astrojs/vue`
- `@astrojs/svelte`
- `@astrojs/solid-js`
- `@astrojs/preact`
- `@astrojs/alpinejs`

## Content integrations

- `@astrojs/mdx`
- `@astrojs/markdoc`

## Platform/utility integrations

- `@astrojs/sitemap`
- `@astrojs/partytown`
- `@astrojs/db`

## Adapters

- `@astrojs/cloudflare` (Astro 6: adapter v13+)
- `@astrojs/netlify`
- `@astrojs/vercel`
- `@astrojs/node`

## Installation

Use `astro add` where available:

```bash
npx astro add react
npx astro add cloudflare
npx astro add mdx sitemap
```

## Upgrade

```bash
npx @astrojs/upgrade
```
